# Kaimaison Berlin (Laravel  Vite)

## Purpose
Website is for a restaurant in Kreuzberg Berlin
## What I am doing
- Install Laravel 11 , MVC Structure
- Create models: Post, User, Order
- Integration contact form & Email notification
- Create Landing Page, home page, about, menu page, news, contact, mobile version
